# Personal Finance Analyzer (C++)
The program allows users to add income and expense details, view all transactions, calculate total income and expenses, and analyze savings. It also provides simple financial suggestions based on the user's spending habits.

## Features
- Add income and expense transactions with details such as date, category, amount, and description
- Display all financial transactions in an organized format
- Calculate total income, total expenses, and net savings
- Provide financial suggestions based on spending patterns
- Simple menu-driven interface for easy interaction

## OOP Concepts Used
- Classes and Objects
- Constructors
- Inheritance
- Polymorphism
- Pointers

## Tech Stack
C++ (Object-Oriented Programming)
